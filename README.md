# My First Kiro

A project bootstrapped and managed with [Kiro](https://kiro.dev), an AI-powered IDE.

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or later recommended)
- [Git](https://git-scm.com/)
- [Kiro IDE](https://kiro.dev)

### Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/rajivdasgupta/my-first-kiro.git
   cd my-first-kiro
   ```
2. Open the project in Kiro

## Project Structure
```
.kiro/
  hooks/          # Agent hooks for automated actions
  steering/       # Project standards and guidelines
```
